# Edgeflix: a Netflix clone

## Features: 

- Signin/signup flow
- Available series list for each user
- Filter by genre, cast, or title
- Movies details on click
- You can:
    - add a movie to your list
    - give a rating from 1 to 5
    - hide a movie
- Responsive layout 
- Video in autoplay as homepage hero
